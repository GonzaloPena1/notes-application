# Notes application
Full Stack Notes Application using Node and Express

Note-Taking Application that allows users to:

Create new notes
Read existing notes
Update notes
Delete notes

[Live website](https://notes-application-4skb.onrender.com/)

## Technologies
HTML, CSS, Javascript, Node.js and Express.js
